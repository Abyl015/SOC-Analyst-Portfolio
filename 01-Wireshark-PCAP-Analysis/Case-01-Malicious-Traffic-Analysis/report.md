# Case 01 - Malicious Traffic Analysis

## Overview

This case demonstrates the analysis of suspicious network traffic using Wireshark.

The purpose of the investigation was to identify a potentially infected internal host, analyze DNS and HTTP traffic, extract indicators of compromise, and determine whether the observed activity could be related to malware communication or command-and-control behavior.
